# XGBoost-Homework

我们使用了遗传算法来求解最优参数。将数据分为 80% 训练集，20% 验证集。结果实验，发现各个参数取如下值时，在测试集上RMSE可以达到3.656079（此时，迭代次数为1191）。

> 
> 'objective': 'count:poisson',
> 
> 'max_depth': 3,
> 
> 'eta': 0.04375073666234408,
> 
> 'gamma': 0
> 


### baseline.py:　	
> 训练模型。80% 训练集，20% 验证集。 <br>
> 结果在 1191 轮达到最优，test-rmse 为 3.656079。
